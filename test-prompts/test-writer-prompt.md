# Test Task: Create Hello World Script

You are tasked with creating a simple Python hello world script.

## Requirements

1. Create a file called `hello.py` in the root directory
2. The script should print "Hello from Agent Cube!" when run
3. Add a docstring explaining what the script does
4. Make the script executable

## Implementation Steps

1. Create `hello.py` with proper structure
2. Add shebang line for Python 3
3. Add docstring
4. Implement the print statement
5. Test that it works

## Final Steps - CRITICAL

After completing all tasks:

1. **Commit your changes:**
   ```bash
   git add -A
   git commit -m "Task: test-hello - Create hello world script
   
   - Created hello.py with proper structure
   - Added docstring and shebang
   - Made by Writer"
   ```

2. **Push to your branch:**
   ```bash
   git push origin writer-sonnet/test-hello
   ```

3. **Verify push succeeded:**
   ```bash
   git log origin/writer-sonnet/test-hello..HEAD
   # Should show no unpushed commits
   ```

⚠️ **CRITICAL**: The judge panel needs your pushed changes to review!

## Success Criteria

- [ ] hello.py exists
- [ ] Script prints correct message
- [ ] Has proper docstring
- [ ] Is executable
- [ ] All changes committed and pushed
