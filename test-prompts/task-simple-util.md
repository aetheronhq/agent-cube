# Task: Add String Utility Functions

Create a simple TypeScript utility module with string helpers.

## Requirements

1. Create `src/utils/string-utils.ts`
2. Implement 3 functions:
   - `capitalize(str)` - Capitalize first letter
   - `truncate(str, maxLength)` - Truncate with ellipsis
   - `slugify(str)` - Convert to URL-safe slug
3. Add tests for all functions
4. Export from `src/utils/index.ts`

## Acceptance Criteria

- All functions type-safe
- Tests pass
- No external dependencies
- Simple, readable code
