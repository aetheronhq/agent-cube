### **Opening** (10 seconds)

### **Launch Command** (5 seconds)
```
I'm giving this agent the Web UI requirements

```

### **Prompter Appears** (30 seconds)
- "First, a Prompter agent reads the requirements"
- "It's thinking about the architecture"
- "Notice it's reading the existing codebase"
- "Understanding the current terminal implementation"
- "Figuring out how to translate this to a web UI"

### **Dual Writers Launch** (1 minute)
**When thinking boxes appear:**
- "Now two agents propose different approaches"
- "Writer A (Sonnet) - notice it's thinking about simplicity"
- "Writer B (Codex) - focusing on completeness"
- "They're both creating planning docs and task files"
- "Different architectural choices emerging..."

**When tool calls show:**
- "Writer A creating files... minimalist approach"
- "Writer B creating files... more comprehensive"
- "Both valid, both different - that's the point"

### **Judge Panel** (1 minute, speed up)
- "Now 3 independent judges review both plans"
- "Judge 1 checking KISS compliance"
- "Judge 2 analyzing completeness"
- "Judge 3 verifying production-readiness"
- [Speed through most of this]

### **Decision** (30 seconds)
**When cube decide shows:**
- "Aggregating the votes..."
- "Winner: [A or B] - [score] vs [score]"
- "Next: Synthesis to combine best of both"
- OR "Clear winner, moving to peer review"

### **Synthesis** (if needed, 30 seconds)
- "Now the winner addresses judge feedback"
- "Taking the best parts of the losing approach"
- "This is where quality compounds"

### **Result** (20 seconds)
- "And there we have it"
- "Complete architecture plan: planning/web-ui.md"
- "6 task files ready to execute"
- "From requirements to executable plan in [X] minutes"
- "Zero human planning time"

### **Closing** (10 seconds)
- "This is how we work now"
- "Agents plan, agents build, judges review"
- "Humans validate and merge"
- "Let's see what they created..."

**[Quick peek at one task file, scroll through]**
